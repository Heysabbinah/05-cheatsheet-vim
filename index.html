The BASICS
Cursor Movement
  • h: Move left
  • j: Move down
  • k: Move up
  • l: Move right
  • w: Jump by start of word
  • W: Jump by word
  • e: Jump to end of word
  • 0: Start of line
  • $: End of line
  • w: Jump by start of word
  • :w Save the file but don't exit
  • :wq Save and quit
  • :q Quit without changes
  • :q! Quit with changes
  • ctrl+ws: Split window
  • ctrl+ww: Switch between windows
  • ctrl+wv: Split window verticaly
  • yy: copy a line
  • yw: copy a word
  • p: Put after the cursor
  • P: Put before the cursor
  • dd: Cut a line
  • dw: Cut a current word
  • x: Cut a current character
